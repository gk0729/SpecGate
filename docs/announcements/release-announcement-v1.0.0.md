# SpecGate v1.0.0 Release Announcement

## 繁體中文（可直接貼文）

今天正式發布 **SpecGate v1.0.0**。

SpecGate 是一份面向 AI 與自動化代碼生成場景的「通用組件接口標準規範」範例，目標是降低組件對接歧義、減少 AI 生成代碼時的接口偏差，並提升跨工具協作的一致性。

### 本版重點

- 發布通用規範（保留完整 L0-L6 多層模型）
- 提供雙語文檔（繁中 / English）
- 採用 MIT 授權，方便社群採納與擴充
- 提供基本治理文件（CONTRIBUTING）

### 核心文件

- 規範（繁中）：`docs/specs/ai-component-interface-standard-example.zh-TW.md`
- 規範（English）：`docs/specs/ai-component-interface-standard-example.en.md`

歡迎 Issues / PR 一起完善標準，推動跨 Agent、插件、SDK、CLI 的可互操作規範。

## English (Ready to Post)

We are excited to announce **SpecGate v1.0.0**.

SpecGate is a public, generic example standard for AI-oriented component interfaces. It aims to reduce integration ambiguity, lower interface mismatch in AI-generated code, and improve interoperability across tools and ecosystems.

### Highlights

- Published a generic interface standard with a full L0-L6 layered model
- Shipped bilingual documentation (Traditional Chinese / English)
- Released under MIT for broad adoption and extension
- Included basic governance docs (CONTRIBUTING)

### Core Documents

- Spec (zh-TW): `docs/specs/ai-component-interface-standard-example.zh-TW.md`
- Spec (en): `docs/specs/ai-component-interface-standard-example.en.md`

Issues and PRs are welcome to co-evolve a practical interoperability standard for agents, plugins, SDKs, and CLIs.
